Context
This is a data set of information, images, and ratings for Shadowverse cards.
Card ratings were scraped from Gamewith and card information and images were scraped from shadowverse_portal.
https://shadowverse.gamewith.jp/
https://shadowverse-portal.com/?lang=ja

Content
・See the column names in shadowverse_dataset.csv for the content of the data.

Note
・Data up to Fortune's Hand (except for the additional cards of Fortune's Hand)
・Cards with leader skins, re-recorded cards, cards with different patterns, etc. are not included in the table data.
For these image paths, only the original designs are associated with them. (ex.Imperial Dragoon and Imperial Dragoon Ardos are also stored in the table data, but only the originals.)
・The csv files are created in utf-8; you may lose some information if you open them in shift-jis. If you want to open them in Excel, please import them in utf-8.
・If the card is rated null, it has not yet been rated. If the card has a "-" rating, it cannot be used in that format (rotation).

Incase you find any discrepancy in the data, please contact the contributors.


【日本語】
概要
Shadowverse（シャドウバース）のカードの情報・画像・評価のデータ集です。
カードの評価はGamewithより、カード情報・画像はshadowverse_portalよりスクレイピングしています。

内容
データの内容については、shadowverse_dataset.csv のカラム名を参照してください。

Note
・FOHまでのデータを収集しました(FOHのアディショナルカードを除く)
・リーダースキン付きカード、再録カード、図柄の異なるカード等はテーブルデータには含まれません。
・これらのイメージパスについては、オリジナルの図柄のみが紐づいています。(ex.インペリアル・ドラグーン、インペリアル・ドラグーン・アルドスもテーブルデータには格納されていますが、オリジナルのみとなります)。
・カードに「null」が付いている場合は、まだレーティングされていません。カードのレーティングが「-」の場合、そのフォーマット（ローテーション）では使用できません。
・csv ファイルは utf-8 で作成されていますので、shift-jis で開くと情報が失われる可能性があります。エクセルで開く場合は utf-8 でインポートしてください。

データに不備があった場合は、投稿者までご連絡ください。
